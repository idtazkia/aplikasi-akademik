Ketentuan Foto Ijazah

Ikhwan

1. Menggunakan jas almamater Institut Tazkia (ketika di foto logo Institut Tazkia harus kelihatan)

2. Menggunakan kemeja warna putih;

3. Menggunakan dasi panjang dengan motif dan warna yang bebas namun formal;

4. Untuk hasil yang bagus agar dapat di Studio Foto.

Akhwat

1. Menggunakan jas almamater Tazkia (ketika di foto logo Tazkia harus kelihatan);

2. Menggunakan jilbab warna putih;

3. Untuk hasil yang bagus agar dapat di Studio Foto.

Ketentuan jenis foto

1. Latar belakang berwarna biru tua;

2. Resolusi ketajaman foto minimal 600 pixel;

3. Ukuran 3x4;

4. Ukuran File yg diupload minimal 1MB;

5. Tidak boleh terlihat gigi ketika senyum;

6. Tidak foto menggunakan kacamata.
